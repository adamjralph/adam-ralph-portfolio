## Requirements

Node.js - v16.12.0

## Installation

```bash
$ git clone git@github.com:adamjralph/adam-ralph-portfolio.git
$ cd adam-ralph-portfolio
$ npm install
$ astro dev
```

### Open browser at:
```bash
http://localhost:3000
```
